== Description ==

Briefcasewp Extras adds more features and functionality to the Briefcase Elementor Widgets. Giving users the option to manage and use easily the bew templates and bew blocks. 

This plugin is meant to be an add-on to Briefcase Elementor Widgets and Elementor Pro page builder as it�s not a standalone plugin.

### Features

* Bew Importer: Import all your Bew Templates and blocks with a simple button.
* Bew Templates: Insert your Bew Templates from the Elementor Pop up.
* Briefcase template type: Manage and control your bew templates on the elementor templates edit page. 

== Installation ==

= Minimum Requirements =
* WordPress 4.9 or greater
* PHP version 5.4 or greater
* MySQL version 5.0 or greater

= Installation Instructions =
- Make sure that you have installed Briefcase Elementor Widgets and Elementor Pro Page Builder. This is not a stand-alone plugin.
- Install the plugin through the WordPress plugins screen directly or download the plugin and upload it to the plugin folder: /wp-content/plugins/.
- Activate the plugin through the installation screen or the �Plugins� screen in WordPress
- You can find Briefcasewp Extras on your wordpress menu sidebar.

== Frequently Asked Questions ==

= Is this a standalone Plugin? =

No. You cannot use Briefcasewp Extras by itself. It is a plugin for Briefcase Elementor Widgets and Elementor Pro.

= Does it work with any WordPress theme? =

Yes. It will work with any WordPress theme that is using Elementor Pro as a page builder.

= Will this plugin slow down my website speed? =

Briefcasewp Extras is light-weight and you can also use only the options you want to use on your website for faster performance.
